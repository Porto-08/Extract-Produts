# Introdução - Extract Products
<p>Extração de dados do Open Food Facts por meio de arquivos para persistência dos dados em um banco de dados MongoDB.</p>

# Instalação
```bash
$ yarn
```

# Iniciar Projeto
```bash
$ yarn start
```
